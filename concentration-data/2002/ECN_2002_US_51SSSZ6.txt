UNIVERSE DESCRIPTION
The universe of this file is all establishments of firms with
payroll. This universe includes all establishments classified in
Information (sector 51) in 2002. These data supersede data released
in the Industry Series reports from the 2002 Economic Census.
DATA ITEMS
This file contains data on:

  Establishments
  Receipts
  Receipts of largest firms as a percent of total receipts
  Annual payroll
  First-quarter payroll
  Paid employees

GEOGRAPHIC COVERAGE
The data are shown at the U.S. level.
INDUSTRY DETAIL
The data are shown at selected 2- through 7-digit NAICS code
levels.
OTHER VARIABLES IDENTIFYING RECORDS
Each record includes CONCENFI which is the three-digit code
representing concentration by largest firms categories (including
all firms, 4 largest firms, 8 largest firms, 20 largest firms, and
50 largest firms).
RELEASE SCHEDULE
The data in this file correspond to the Subject Series,
Establishment and Firm Size (Including Legal Form of Organization)
report (EC02-51SS-SZ) that will be released in December 2005.
RELATED DATA FILES
EC0251SSSZ1 through EC0251SSSZ5 and EC0251SSSZ7 present data by
employment and receipts size for establishments and firms, number
of single unit and multiunit firms, and legal form of organization
for the United States.
CORRESPONDENCE TO PDF REPORTS
These data are exactly the same as are shown in the PDF-format
Subject Series, Establishment and Firm Size (Including Legal Form
of Organization) report (EC02-51SS-SZ). This report will also be
released on the Census Bureau's Internet site at
www.census.gov.
DATA LIMITATIONS
There are no special limitations to these data.
FILE SEQUENCE
Data are presented by firm concentration in ascending NAICS code
sequence.
